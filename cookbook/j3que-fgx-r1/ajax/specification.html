This is a cookbook prototype of a multi-window user experience.

The windows are intended to provide the 2D user interface that control aspects of an underlying 3D display.

Requirements include the following:

  • There is a selection of themes available. These themes are user-selectable and the display updates as soon as a new theme is seleected
  • The windows may be freely moved and resized
  • The windows may be minimized using the '-' icon and closed using the 'x' icon
  • All settings - themes, window positions and sizes as well as particular app setings - may be saved for future use and sent as permalinks
  • Multiple sesssions may be open at the same time - each with different settings
